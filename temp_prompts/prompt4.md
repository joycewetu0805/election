Prompt 4 : Le Dashboard Admin - Gestion du CMS et Candidats

Objectif : Créer l'interface d'administration en React avec Tailwind CSS.

Fais une page avec un layout Dashboard (Sidebar à gauche, contenu à droite).

Section Configuration : Un formulaire pour modifier la table site_settings (Titre, Sous-titre, Informations). Les modifications doivent se sauvegarder en temps réel dans Supabase.

Section Candidats : Un tableau CRUD pour ajouter, modifier ou supprimer des candidats. Ajoute un bouton (style étoile) pour définir un candidat comme "Phare" (cela doit passer is_featured à true pour ce candidat, et à false pour tous les autres).
