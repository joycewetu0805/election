Prompt 7 : La Page de Confirmation

Objectif : Créer la page de fin de parcours (/confirmation).
Cette page est une voie sans issue pour l'électeur. Affiche un message de succès propre : "Merci pour votre participation ! Votre vote a été pris en compte."
Affiche le nom du candidat pour lequel l'utilisateur a voté (récupéré via Supabase).
Ajoute un gros bouton "Se déconnecter".
Assure-toi que le composant empêche le retour en arrière via le navigateur (History API) pour ne pas revoir la page de vote.
