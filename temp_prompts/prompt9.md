Prompt 9 : Design System (Gris Nardo, Minimalisme & Responsive)

Objectif : Appliquer une charte graphique premium et responsive à l'ensemble du projet (Admin, Vote, Résultats).

Palette de couleurs : Utilise exclusivement du Blanc (#FFFFFF), du Noir pur (#000000), du Gris anthracite et le Gris Nardo (#6B7075 ou #C0C0C0 selon la nuance souhaitée) pour les surfaces secondaires.

Mode Clair / Sombre : Implémente un switch de thème utilisant Tailwind dark:.

Mode Clair : Fond blanc cassé, textes noir profond, jauges gris nardo.

Mode Sombre : Fond noir mat, cartes gris foncé, textes gris très clair.

Responsivité : >    - Mobile : Liste sur une seule colonne, textes larges pour lisibilité.

Tablette/Desktop : Grille de candidats (2 ou 3 colonnes) pour la page de vote, mais reste en liste verticale pour les résultats (façon "Classement").

Typographie : Utilise une police sans-serif très "institutionnelle" (type Inter ou Geist Sans) avec beaucoup d'espacement (tracking) pour les titres.
